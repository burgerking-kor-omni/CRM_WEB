A. 팝업을 호출하여 부모창과 팝업 창과의 데이타 전송방법.

	1. 부모창
	   팝업창으로 부터 데이타를 받아오는 리스너 Function 을 작성하여
	   팝업창 호출시 해당 리스너 Function 을 넘겨준다.
	   
	   ex) 
	    // 팝업창 데이타 수신 리스너
	    var listner = function(data) {
			$('#EXAMPLE_ID').val(data.ID);
			$('#EXAMPLE_NAME').val(data.NAME);
		};
	    
	    // 팝업 호출
	    uxl.openWindow('PopupName', 'ScreenId', popupOption, listner);
	
	
	2. 팝업창
	   부모창에게 데이타를 넘겨주기 위해 uxl.pushToParent 함수를 호출한다.
	   
	   ex) 
		// 부모창에게 전송할 데이타
		var data = {
			ID: selectedId,
			NAME: selectedName
		}
		
		// 부모창에게 데이타 전송
		uxl.pushToParent(data);
	
	
	*. 부모창과 팝업창과의 연계는 받듯이 위와 같이 리스너-통지 패턴을 사용한다.
	   위와 같은 패턴으로 해결이 안되는 케이스가 발견 될 경우 연구소로 해당 케이스를 리포팅 바람.
	
	
B. 모달 팝업을 호출하여 부모창과 모달창과의 데이타 전송방법.

	1. 부모창
	   팝업창으로 부터 데이타를 받아오는 리스너 Function 을 작성하여
	   팝업창 호출시 해당 리스너 Function 을 넘겨준다.
	   
	   ex) 
	    // 팝업창 데이타 수신 리스너
	    var listner = function(data) {
			$('#EXAMPLE_ID').val(data.ID);
			$('#EXAMPLE_NAME').val(data.NAME);
		};
	    
	    // 모델창 호출
	    uxl.openModalWindow('PopupName', 'ScreenId', popupOption, listner);
	
	
	2. 모달창
	   부모창에게 데이타를 넘겨주기 위해 uxl.pushToParent 함수를 호출한다.
	   
	   ex) 
		// 부모창에게 전송할 데이타
		var data = {
			ID: selectedId,
			NAME: selectedName
		}
		
		// 부모창에게 데이타 전송
		
		(X) uxl.pushToParent(data); <-- 이렇게 쓰는게 아니고 아래처럼 쓴다.
		
		(O) window.returnValue = data; <--이렇게 쓴다.
